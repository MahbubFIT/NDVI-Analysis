# NDVI Analysis for Saxon Switzerland National Park

## Installation

To run this code, you need to install the required dependencies. Use the following command:

```bash
pip install -r dependencies.txt
```

Ensure you have Python installed (preferably version 3.8 or higher).

## Running the Code

1. Make sure all the required data files (TIFF images and shapefile) are correctly placed in the specified directories.
2. Open the Jupyter Notebook (`Projectfile `) in Jupyter Lab or Jupyter Notebook.
3. Execute the cells in sequence to load the data, compute NDVI, and visualize the results.

## Notes

- The paths in the code should be updated to match your local file structure.
- Ensure you have sufficient memory to handle raster data processing.
